# Projeto de Dashboard de Vendas X-box - Bootcamp DIO

## Entendendo o Desafio

O objetivo deste desafio é criar um dashboard de vendas, com foco na organização e visualização de dados. O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

## Sobre o Banco de Dados Utilizado:
As informações deste dashboard foram retiradas através do banco de dados os quais tinham as seguintes colunas sobre as vendas da plataforma do Xbox:
* Name;
* Gender;
* Plan (Plano escolhido tendo como categorias Standart, Core e Ultimate);
* Start Date ou data de início;
* Auto Renewal (Auto Renovável);
* Subscription Price;
* Subscription Type (Monthly, Quartely e Annual);
* EA Play Seanson Pass (Se incluia o passe da EA);
* EA Season Pass Price;
* Minecraft Season Pass (Se incluia o passe do Minecraft);
* Minecraft Season Pass Price;
* Coupon Value;
* Total Value

O objetivo deste desafio foi aplicar os conceitos e ferramentas de tabela dinâmica, gráficos e estilos para elaborar um dashboard contendo a análise das vendas da plataforma X-box, conforme a base de dados disponibilizada.
